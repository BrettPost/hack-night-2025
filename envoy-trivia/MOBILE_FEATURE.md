# 📱 Mobile Participation Feature

## How It Works

1. **Main Screen** (http://localhost:3000)
   - Shows trivia questions on the big screen
   - QR code in bottom-left corner
   - Answer counter appears below timer when people answer

2. **Mobile Screen** (http://localhost:3000/mobile)
   - Scan QR code with your phone to join
   - See the same question on your phone
   - Tap to submit your answer
   - Get instant confirmation

## Quick Start

```bash
# Start the app
npm run dev

# Open on your computer
http://localhost:3000

# Scan QR code with your phone OR
# Go to this URL on your phone (use the Network URL from terminal):
http://10.80.1.246:3000/mobile
```

## Features

✅ Real QR code generation
✅ Live answer counting
✅ Mobile-optimized interface
✅ Answer tracking per question
✅ Auto-reset between questions
✅ Real-time updates

## Usage

1. Display main screen on TV/projector
2. Audience scans QR code with phones
3. They see questions and can answer
4. Main screen shows total number of answers
5. Counter resets automatically for each new question
